# AnonChat Frontend

Pure HTML, CSS, Vanilla JavaScript frontend.

## Files
- `index.html` — Home page (create/join room)
- `room.html` — Chat room page
- `admin.html` — Global admin dashboard
- `js/config.js` — Backend URL configuration
- `js/utils.js` — Shared utilities (toast, theme, etc.)
- `js/home.js` — Home page logic
- `js/room.js` — Room chat logic
- `js/admin.js` — Admin dashboard logic

## Configuration
Edit `js/config.js` to set the backend URL.

## Usage
The frontend is automatically served by the backend at `http://localhost:3000`.
You can also serve it separately with any static file server.
